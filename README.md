# Online Restaurant Management System 🍽️

A full-stack web application developed as a final project, focusing on modern backend architecture and seamless user experience.

## 🚀 Overview
This project is an **Online Restaurant Website** built with **ASP.NET Core**. It aims to provide an end-to-end solution for restaurant management, from menu browsing to order handling.

## 🛠️ Technologies
- **Backend:** C#, ASP.NET Core.
- **Data:** Entity Framework Core, SQL Server.
- **Frontend:** HTML, CSS, JavaScript (Razor Pages/MVC).

## ✨ Key Features
- **Dynamic Menu:** Users can browse categories and view real-time updates.
- **Order Management:** Optimized backend logic for handling customer requests.
- **Admin Dashboard:** Full CRUD operations for menu and order control.
